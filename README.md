# 📝 tryhackme-writeups

Write-Ups for TryHackMe rooms (Walkthrough & CTFs) written in English.

https://tryhackme.com/

## Disclaimer

> Writeups should have a link to TryHackMe and not include any passwords/cracked hashes/flags

- Sort the rooms alphabetically
- For reference links use your `pseudo-roomURLname` (eg. `zombrax-learnowaspzap` for https://tryhackme.com/room/learnowaspzap)

## Submissions

If you want to incorporate your own writeup to solve a room you can do it through a pull request.
